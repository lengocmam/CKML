# Jupyter Notebooks cho phân tích (optional)

Folder này có thể chứa:
- `exploratory_analysis.ipynb` - Phân tích khám phá dữ liệu
- `feature_engineering.ipynb` - Thử nghiệm features
- `model_experiments.ipynb` - Thử nghiệm models khác nhau
- `visualization.ipynb` - Vẽ charts

**Hiện tại: Không bắt buộc**

App chính chạy được mà không cần notebooks.
